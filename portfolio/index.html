Back2Front框架

  • Node.js

2015/4 至今

基于Express和XTemplate开发，起初只是一个前后端中间平台，后来进一步改良为模块化开发框架，有以下特性：

  • 通过XTemplate的「parse」或「include」指令就可以把一个模块的HTML、CSS和JS都引用进来；
  • 支持对项目内的静态资源进行预处理；
  • 支持前后端同构。

此外，还有配套的CLI工具「Back2Front-CLI」，可用于分析依赖和构建项目。

本框架1.0之前的版本应用于美黛拉Web端产品；1.0及以后的版本应用于贝聊官网。

  • Javascript

2010/1 至 2010/10

为网易门户网站常规专题制作而开发的辅助系统，可以在所见即所得的界面中，通过组合模块快速创建专题页面。

专题页面的布局与模块组合形式以JSON格式存储，并以模板注释的形式保存在最终生成的页面代码中，以便下次编辑时直接加载。
